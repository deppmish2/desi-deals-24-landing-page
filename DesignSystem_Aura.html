DesiDeals24 Design System: "Aura"

       
           

Design Principles:

           
                   
  • Clarity: The interface is intuitive, legible, and predictable.
    •                
  • Deference: The UI never competes with the content (the deals).
    •                
  • Efficiency: User flows are streamlined and focused.
    •            
       
       

1. Foundations

       

The "Aura" system is founded on a clean, neutral palette that elevates the content. A single, vibrant blue is used exclusively for interactive elements, creating an unambiguous user experience.

Typografie, Layout & Spacing

       

The established guidelines for the Inter typeface, the 12-column responsive grid, and the 8px spacing system are fundamental to this clean aesthetic and remain unchanged.

       

2. Components (Example: Button)

       

Button styles are simplified to three clear variants, each with a distinct hierarchical role.

                                                                                                                                                                                                                                                                                                                                                                                                                                                 
VariantStyleUsage
PrimarySolid Action Blue background with white text. The single most important action on a screen (e.g., "Go to Deal").
Secondary Fill Gray background with Action Blue text. Secondary actions like applying filters or navigating.
Tertiary (Ghost)Transparent background with Action Blue text.Low-emphasis actions like "Cancel" or "Clear All."
       

3. Patterns & Flows

       

User flows are self-evident due to the consistent application of the interactive Action Blue color against the neutral background, guiding the user's eye naturally.

               

4. Design Tokens
